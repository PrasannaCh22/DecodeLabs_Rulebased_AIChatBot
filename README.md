# 🤖 Rule-Based AI Chatbot

## 📌 Project Overview

This project was developed as part of my Artificial Intelligence Internship training.

The objective of this project is to create a simple Rule-Based AI Chatbot using Python. The chatbot interacts with users by responding to predefined inputs through conditional logic instead of machine learning.

This project focuses on understanding the fundamentals of Artificial Intelligence including decision-making, control flow, and user interaction.

---

## 🎯 Project Objective
To design and develop a chatbot that:

- Accepts user input
- Identifies predefined commands
- Generates suitable responses
- Runs continuously until the user exits

---

## 🚀 Features

✔ Handles greeting messages  
✔ Supports multiple user inputs  
✔ Uses conditional (if-else) logic  
✔ Runs in a continuous loop  
✔ Handles exit commands  
✔ Provides default responses for unknown inputs
